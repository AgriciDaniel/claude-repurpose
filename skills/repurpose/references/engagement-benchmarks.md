# Per-Platform Engagement Benchmarks (2026)

Data-backed benchmarks for setting realistic expectations and evaluating repurposed content quality.

## Twitter/X

| Metric | Value |
|--------|-------|
| Average engagement rate | 0.05% (low bar, high variance) |
| Threads vs single tweets | +63% more impressions |
| Sweet spot tweet length | 71-100 characters (+17% engagement) |
| Reply weight vs like | 27x |
| Conversation weight vs like | 150x |
| Critical window | First 30-60 minutes determines wider distribution |
| Video tweets vs text-only | 10x engagement |
| Visibility half-life | Tweets lose 50% visibility every 6 hours |

**Key takeaway:** Twitter rewards conversation above all. A post with 10 thoughtful replies outperforms one with 100 likes. Threads dramatically extend reach and should be the default long-form format.

## LinkedIn

| Metric | Value |
|--------|-------|
| Average engagement rate | 2.0-3.5% (varies by follower count) |
| 0-2K followers | 4.0-6.0% engagement |
| 2K-10K followers | 2.5-4.0% engagement |
| 10K-50K followers | 1.5-2.5% engagement |
| Critical window | First 60-90 minutes (only 5% of underperformers recover) |
| Most valuable signal | Saves (5x the value of a like) |

### Engagement by Format

| Format | Engagement Rate |
|--------|----------------|
| PDF carousels | 6.60% (highest) |
| Multi-image posts | 6.60% |
| Native video | 5.00% |
| Single image | 4.85% |
| Polls | 4.4-8.9% |
| Text-only | 4.00% |

**Key takeaway:** PDF carousels and polls dominate LinkedIn. The first 90 minutes are make-or-break; post when your audience is online and engage immediately with every comment.

## Instagram

| Metric | Value |
|--------|-------|
| Carousel engagement | 1.92-2.33% |
| Mixed media carousel (image+video) | 2.33% |
| Image-only carousel | 1.80% |
| Reel engagement | 1.5-10% (highly niche-dependent) |
| "Swipe left" prompt effect | +10% engagement |
| Optimal carousel length | 7-10 slides (>60% completion rate) |

### Algorithm Priority (highest to lowest)
1. Watch time
2. Sends and shares
3. Saves
4. Likes

**Key takeaway:** Mixed-media carousels (combining images and short video clips) outperform pure image carousels. Always include a "swipe left" prompt. Saves and shares matter more than likes.

## Facebook

| Metric | Value |
|--------|-------|
| Average organic reach | ~0.06% |
| Feed composition | 50% from accounts you don't follow (AI recommendations) |
| Highest priority format | Short-form video (Reels) |
| Most powerful signals | Saves + DM shares |

### Poll Performance
- Polls with images: significantly higher engagement than text-only polls
- Image/GIF enhancement recommended for all poll posts

**Key takeaway:** Facebook organic reach is minimal unless you hit the AI recommendation engine. Short-form video and shareable polls with images are the viable paths.

## YouTube Community

| Metric | Value |
|--------|-------|
| Highest engagement format | Polls |
| Second highest format | Images + GIFs |
| Optimal posting frequency | 2-4 posts/week |
| Diminishing returns | Beyond 4 posts/week |

**Key takeaway:** Community posts keep subscribers engaged between uploads. Polls consistently outperform other formats. Do not over-post; 2-4 per week is the sweet spot.

## Reddit

| Metric | Value |
|--------|-------|
| Primary success metrics | Upvotes + comment depth |
| Discussion quality vs view count | Quality wins |
| Self-promotion tolerance | 10% maximum of total activity |
| Community norms | Vary by subreddit (always read rules first) |

**Key takeaway:** Reddit punishes overt self-promotion. Repurposed content must be genuinely valuable and community-native. Discussion quality matters more than raw views.

## Newsletter/Email

| Metric | Good | Excellent |
|--------|------|-----------|
| Open rate | 20-25% | 30%+ |
| Click-through rate | 2-5% | 5%+ |

### Subject Line Optimization

| Tactic | Effect |
|--------|--------|
| Numbers in subject line | +57% opens |
| Personalized subject | +14.68% opens |
| Preview text length | 40-90 characters |
| Preview text rule | Complement the subject, never repeat it |

**Key takeaway:** Subject line is everything. Numbers and personalization measurably improve open rates. Preview text is an underused second hook that should add context, not echo the subject.
