# Platform Specifications

Comprehensive specs for all supported platforms. Always consult before generating any platform output.

## Twitter/X

| Spec | Value |
|------|-------|
| **Character limit** | 280 (standard), 25,000 (Premium) |
| **Sweet spot** | 71-100 characters for highest engagement |
| **Thread length** | 8-12 tweets optimal (+63% impressions vs single tweet) |
| **Images** | 1600x900 (16:9), up to 4 per tweet, max 5MB each, JPG/PNG/GIF |
| **Video** | Up to 2min 20s (140s), 1920x1200 max resolution |
| **Algorithm weight** | Reply = 27x a like; Conversation = 150x a like |
| **External links** | Near-zero reach for non-Premium accounts; avoid in tweet body |
| **Hashtags** | 0-2 maximum (more = lower reach) |
| **Post types** | Text, image, video, thread, poll (4 options, 7-day max duration) |
| **Best posting time** | Tuesday 9AM; generally 8-10AM weekdays |
| **Key rule** | Threads massively outperform single tweets. Always offer a thread version. |

## LinkedIn

| Spec | Value |
|------|-------|
| **Post character limit** | 3,000 characters |
| **Best format** | PDF carousel: 6.60% engagement rate (HIGHEST of all formats) |
| **Images** | 1080x1080 (1:1) or 1080x1350 (4:5); safe zone 880x880 for mobile |
| **Document/PDF** | Up to 300 pages, 100MB max; native carousel format |
| **Video** | Up to 10 minutes, 1080x1080 or 1920x1080 |
| **Algorithm weight** | Saves = 5x a like; only comments >10 words count; first 60-90 min critical |
| **External links** | 40% less reach; always place links in first comment, not post body |
| **Hashtags** | 3-5 optimal |
| **Post types** | Text, image, carousel (native), document (PDF), video, poll (4 options, 2-week max), article |
| **Best posting time** | Tuesday-Thursday 8-10AM |
| **Key rule** | First line is the hook. Open with a bold statement or question. Carousels dominate. |

## Instagram

| Spec | Value |
|------|-------|
| **Caption limit** | 2,200 characters (125 visible before "more" truncation) |
| **Carousel slides** | 7-10 optimal, up to 20 max; 1080x1350 (4:5) |
| **Mixed media carousel** | 2.33% engagement vs 1.80% image-only carousel |
| **Reel dimensions** | 1080x1920 (9:16), 30-90s sweet spot, up to 3 minutes |
| **Story dimensions** | 1080x1920 (9:16), 15-second segments |
| **Algorithm weight** | Watch time #1, then sends/shares, then saves |
| **Hashtags** | 3-5 niche hashtags (NOT 30; over-tagging reduces reach) |
| **Post types** | Feed image, carousel, reel, story |
| **Best posting time** | Monday/Wednesday/Friday 11AM-1PM |
| **Hidden gem** | "Swipe left" prompt in caption = +10% engagement (only 5% of creators use it) |
| **Key rule** | First slide of carousel must stop the scroll. Visual-first platform. |

## Facebook

| Spec | Value |
|------|-------|
| **Post limit** | No strict limit (optimal: 40-80 characters for highest engagement) |
| **Images** | 1080x1350 (4:5) best for feed; 1200x628 for link previews |
| **Video** | Up to 240 minutes; all videos auto-classified as Reels |
| **Story** | 1080x1920 (9:16) |
| **Algorithm weight** | Saves + DM shares most powerful; 50% of feed comes from non-followed accounts |
| **Content mix** | 80% value / 20% promotional (strict) |
| **Post types** | Text, image, video/reel, story, poll (up to 10 options), link |
| **Best posting time** | Tuesday 9AM-12PM |
| **Key rule** | End posts with a question. Community-building signals drive reach. |

## YouTube Community

| Spec | Value |
|------|-------|
| **Text limit** | 5,000 characters |
| **Poll options** | Up to 5 options; 65 chars per text option, 36 chars per image option |
| **Images** | 1080x1080 or 1280x1280, max 16MB, JPG/PNG/GIF/WEBP |
| **Frequency** | 2-4 posts per week |
| **Post types** | Text, image, poll, GIF |
| **Best format** | Polls (highest engagement by far) |
| **Key rule** | Polls drive 2-3x engagement vs text posts. Always include one. |

## Skool

| Spec | Value |
|------|-------|
| **Character limit** | No strict limit |
| **Supports** | Text, images, GIFs, short video |
| **Community health** | 20-30% monthly active members = healthy benchmark |
| **Content rotation** | REQUIRED: mix polls, discussions, challenges, questions weekly |
| **Admin responsiveness** | Make-or-break engagement signal; respond within 2 hours |
| **Growth timeline** | 60-90 days to build real community momentum |
| **Key rule** | Gamification matters. Frame posts as challenges or action items. |

## Reddit

| Spec | Value |
|------|-------|
| **Title limit** | 300 characters |
| **Post body limit** | 40,000 characters (text post), or link/image post |
| **Subreddit rules** | Vary per community; always check before posting |
| **Self-promotion rule** | 10% max self-promotional content (90% must be genuine contribution) |
| **Formatting** | Full Markdown supported |
| **Algorithm weight** | Upvotes + comment depth + early velocity |
| **Post types** | Text, image, link, poll, video |
| **Key rule** | NO marketing speak. Be peer-to-peer, evidence-first, genuinely helpful. |

## Newsletter / Email

| Spec | Value |
|------|-------|
| **Subject line** | 40-60 characters optimal; 9 words ideal length |
| **Preview text** | 40-90 characters |
| **Body length** | 200-500 words optimal for newsletters |
| **Email sequence** | 3-email drip: day 0 (value), day 2 (deeper dive), day 4 (CTA) |
| **Open rate benchmark** | 20-25% (good), 25-35% (excellent) |
| **Click rate benchmark** | 2-5% (good), 5-10% (excellent) |
| **Key rule** | Subject line determines 80% of performance. Personalization adds +14.68% open rate. |
